Weather App: Real-Time Weather Updates

WeatherApp is a sleek and modern weather application built using Flutter. The app leverages state management providers to ensure a smooth and responsive user experience, with a focus on real-time weather updates for the user's current location. It also supports dark mode, providing users with a visually appealing interface that adapts to their system preferences.

Project

News App: Dynamic News Application

NewApp is a dynamic news application developed using Flutter, adhering to the MVVM (Model-View-ViewModel) architecture for a structured and maintainable codebase. The app provides users with the latest news articles, fetched from the Google News API. To ensure secure access, Firebase authentication is integrated, allowing users to sign in using their Google account or via email and password. Additionally, the app includes an email verification feature to enhance user security.
